# BookStore Web Page 📚
A simple and responsive BookStore web page built using HTML and CSS. This project showcases a clean user interface for displaying books, categories, featured collections, and book details.
## Features
* Responsive design for desktop and mobile devices
* Attractive homepage layout
* Book categories and featured books section
* Modern CSS styling and hover effects
* Clean and beginner-friendly code structure
## Technologies Used
* HTML5
* CSS3
## Purpose
This project was created to practice front-end web development skills, including page layout design, responsive design techniques, and CSS styling.
